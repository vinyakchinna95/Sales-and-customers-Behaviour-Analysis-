# 📊 Customer Shopping Behavior Analysis

## 📌 Overview
This project demonstrates an end-to-end data analytics workflow, transforming raw data into actionable insights. It includes data preprocessing, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard visualization to understand customer purchasing behavior.

---

## 📂 Dataset
- **File:** `customer_shopping_behavior.csv`  
- Contains customer demographics, transaction details, and behavioral attributes such as purchase frequency and ratings.

---

## 🛠️ Tools & Technologies
- **Python:** Data loading, EDA, preprocessing  
- **SQL (PostgreSQL / MySQL / SQL Server):** Data analysis  
- **Tableau:** Dashboard visualization  
- **Gamma:** Presentation (PPT) creation  

---

## ⚙️ Project Steps

### 1. Data Loading
- Imported dataset using Python  
- Inspected structure and identified missing values  

### 2. Exploratory Data Analysis (EDA)
- Analyzed data distributions and patterns  
- Explored relationships between key variables  

### 3. Data Cleaning
- Handled missing values  
- Standardized column names  
- Removed redundant columns  

### 4. SQL Analysis
- Loaded cleaned data into database  
- Performed queries for:
  - Revenue analysis  
  - Customer segmentation  
  - Category and product insights  

### 5. Dashboard Creation
- Built an interactive dashboard using Tableau  
- Included KPIs and visual insights  

### 6. Reporting & Presentation
- Documented findings in a structured report  
- Created presentation using Gamma
